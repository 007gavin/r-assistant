1. 安装

从 GitHub 安装（推荐）

install.packages("remotes")
remotes::install_github("007gavin/r-assistant")

从本地安装

install.packages("D:/r-assistant/r.assistant_0.3.0.tar.gz",
                 repos = NULL, type = "source")
依赖包

自动安装：httr2 jsonlite rstudioapi shiny miniUI clipr processx

2. 配置 API

library(r.assistant)
assistant_config(provider = "deepseek", api_key = "sk-xxx")
assistant_get_config()  # 查看配置
API Key

保存在本地 ~/.r-assistant/config.json

4. 模型选择

# 查看当前提供商的可用模型
assistant_list_models()
assistant_list_models("openai")    # 指定提供商

# 交互式选择模型（弹出选择界面）
assistant_select_model()

# 命令行直接切换
assistant_set_model("deepseek-coder")
assistant_set_model("gpt-4o")
Viewer 面板中切换

聊天面板顶部有模型下拉选择器，点击即可切换，无需输入命令。

5. 对话问答

assistant_chat("如何画箱线图？")   # 多轮对话
assistant_chat("加上颜色分组")     # 保留上下文
assistant_ask("lapply vs sapply?") # 一次性

6. Viewer 面板聊天

addin_chat()          # 启动聊天（后台运行，Console 空闲）
addin_chat_close()    # 关闭
手动打开 Viewer

addin_chat() 启动后会显示一个 URL（如 http://127.0.0.1:54321）。

如果 Viewer 面板没有自动打开，复制该 URL 并执行：

rstudioapi::viewer("http://127.0.0.1:54321")  # 替换为实际端口号

每次启动端口号不同，请以控制台显示的 URL 为准。

模型选择器

顶部下拉框，点击切换 LLM 模型

历史按钮

时钟图标，查看对话历史

新对话

+ 按钮，清空并开始新对话

插入代码

文件图标，将 AI 代码插入编辑器

7. 历史记录

assistant_history(n = 10)        # 最近 10 条
assistant_history(as_messages = FALSE) # 带时间戳
assistant_clear_history()       # 清空

在 Viewer 聊天面板中，点击右上角时钟图标可直接查看历史，无需输入命令。

8. 代码解释

assistant_explain(code, detail_level = "normal")
# "brief" | "normal" | "detailed"

9. 代码重构

assistant_refactor(code, style = "tidyverse", goal = "readability")
# style: "tidyverse" | "base" | "data.table"
# goal:  "readability" | "performance" | "conciseness"

10. 错误修复

assistant_fix()                         # 自动读取选中代码+错误
assistant_fix(code, error)              # 手动指定

11. 代码与文档生成

assistant_complete("创建正态性检验函数")
assistant_document("my_func <- function(x) x")
assistant_test("add <- function(a, b) a + b")
