1. 安装

从 GitHub 安装（推荐）

install.packages("remotes")
remotes::install_github("007gavin/r-assistant")

从本地安装

install.packages("D:/r-assistant/r.assistant_1.0.3.tar.gz",
                 repos = NULL, type = "source")
依赖包（自动安装）

httr2 jsonlite rstudioapi shiny miniUI clipr

2. 首次配置

交互式设置向导

library(r.assistant)
assistant_setup()     # 引导选择 provider 和输入 API key

手动配置

assistant_config(provider = "siliconflow",
                 api_key  = "sk-xxx")

验证配置

assistant_check()
# Provider: siliconflow
# Model: Qwen/Qwen3.6-27B
# Testing API at: https://api.siliconflow.cn/v1/chat/completions
# [OK] API connection successful!
加载时自动提示

library(r.assistant) 会自动显示当前配置状态。首次使用无配置时会提示 assistant_setup()。

未设置 → 使用 provider 默认地址。

5. Viewer 面板聊天

addin_chat()          # 启动聊天（后台运行，Console 空闲）
addin_chat_close()    # 关闭
手动打开 Viewer

addin_chat() 启动后显示 URL（如 http://127.0.0.1:54321）。

如果 Viewer 面板没有自动打开，复制 URL 执行：

rstudioapi::viewer("http://127.0.0.1:54321")

每次启动端口号不同，以控制台显示为准。

面板功能

模型选择器

顶部下拉框，点击切换 LLM 模型

设置面板

齿轮图标，调节温度、Token、上下文开关

历史按钮

时钟图标，查看对话历史

新对话

+ 按钮，清空并开始新对话

插入代码

文件图标，将 AI 代码插入编辑器

复制按钮

鼠标悬停代码块右上角，一键复制

6. 命令行对话

# 多轮对话（保留上下文）
assistant_chat("如何用 ggplot2 画箱线图？")
assistant_chat("再加上颜色分组")

# 快速提问（一次性，不保留历史）
assistant_ask("lapply 和 sapply 的区别？")

7. 代码解释

assistant_explain("mtcars %>% group_by(cyl) %>% summarise_all(mean)")
assistant_explain()                        # 解释选中的代码
assistant_explain(code, detail_level = "brief")     # 简要
assistant_explain(code, detail_level = "detailed") # 详细

8. 代码重构

assistant_refactor(code, style = "tidyverse", goal = "readability")
# style: "tidyverse" | "base" | "data.table"
# goal:  "readability" | "performance" | "conciseness"

9. 错误修复

assistant_fix()                          # 自动读取选中代码 + 最后错误
assistant_fix(code, error)               # 手动指定

10. 代码与文档生成

assistant_complete("创建正态性检验函数")          # 生成代码
assistant_document("my_func <- function(x) x")  # 生成 roxygen2 文档
assistant_test("add <- function(a, b) a + b")   # 生成 testthat 测试

11. 上下文感知

上下文内容控制参数
会话信息R 版本、OS、工作目录始终包含
已加载包R 包列表include_session
全局变量变量名、类型、维度、大小include_env
当前文档RStudio 打开的文件内容include_document
选中代码编辑器中选中的文本include_selection
assistant_context()                      # 查看当前上下文
assistant_config(context_enabled = FALSE)  # 关闭上下文
上下文压缩

当对话超过上下文窗口时，自动压缩旧消息（保留最近 6 条 + 摘要）。

聊天面板底部显示上下文使用量进度条（绿色 <50%、黄色 50-80%、红色 >80%）。

12. 历史记录

assistant_history(n = 10)               # 最近 10 条
assistant_history(as_messages = FALSE)  # 带时间戳
assistant_clear_history()              # 清空

在 Viewer 聊天面板中，点击右上角时钟图标可直接查看历史。

13. 设置面板

点击聊天面板右上角齿轮图标打开设置：

Temperature

0-2 滑块，越低越确定

Max Tokens

最大回复长度（默认 4096）

Context Awareness

开关，是否自动包含 R 会话上下文

Provider / Model

显示当前配置信息

16. 常见问题

Q: HTTP 400 Bad Request

通常是 model 名称错误或 URL 不匹配。执行 assistant_check() 查看实际请求的 URL。

Q: HTTP 404 Not Found

URL 路径错误。清除 base_url 使用 provider 默认：

assistant_config(base_url = "")
assistant_set_provider("siliconflow")
assistant_check()

Q: HTTP 401 Unauthorized

API Key 无效或过期。重新设置：

assistant_set_key("new-key")

Q: addin_chat() 报错找不到函数

包未正确加载。重启 RStudio 后执行 library(r.assistant)。

Q: Viewer 面板没有自动打开

手动执行：

rstudioapi::viewer("http://127.0.0.1:端口号")

Q: Addins 菜单中看不到 R Assistant

重启 RStudio，然后 Tools → Addins → Browse Addins 搜索。

Q: 如何在多人环境中使用？

每人首次执行 assistant_setup() 配置自己的 API Key。
