r.assistant

R AI 编程助手 - 在 RStudio Viewer 面板中运行的 AI 辅助编程工具

v0.2.2 Changelog

  • Chat runs in background process - Console completely free
  • Opens in RStudio Viewer pane (alongside Files/Plots/Help)
  • Dark theme matching RStudio IDE
  • Enter to send, Shift+Enter for newline
  • Insert code into editor with one click

1. 安装

# 从本地安装
install.packages("D:/r-assistant/r.assistant_0.2.2.tar.gz",
                 repos = NULL, type = "source")

# 或从 GitHub 安装
# install.packages("remotes")
# remotes::install_github("0101123/r-assistant")
依赖包

自动安装：httr2、jsonlite、rstudioapi、miniUI、shiny、clipr、processx

2. 配置 API

library(r.assistant)

# 一步到位
assistant_config(provider = "deepseek", api_key = "sk-xxx")

# 或分别设置
assistant_set_provider("deepseek")
assistant_set_key("sk-xxx")

# 查看当前配置
assistant_get_config()
API Key 存储

保存在本地 ~/.r-assistant/config.json，不会上传。

4. 对话问答

# 多轮对话（保留上下文）
assistant_chat("如何用 ggplot2 画箱线图？")
assistant_chat("再加上颜色分组")       # 自动保留上文

# 快速提问（一次性，不保留历史）
assistant_ask("lapply 和 sapply 的区别？")

5. Viewer 面板聊天

v0.2.2 新特性

Chat 运行在后台进程中，Console 完全空闲，可以同时写代码和聊天。界面显示在 RStudio Viewer 面板中。

# 打开 Viewer 面板聊天
addin_chat()

# 关闭聊天（释放后台进程）
addin_chat_close()

使用方式

  1. 执行 addin_chat() 或点击 Addins -> R Assistant Chat
  2. 聊天面板出现在 Viewer 面板位置（右下角，和 Files/Plots 同级）
  3. 可以拖拽 Viewer 标签到左侧面板区域
  4. Console 完全空闲，可以正常输入 R 代码
  5. 输入框中 Enter 发送，Shift+Enter 换行

面板功能

+ 新对话

清空聊天历史，开始新的对话

Insert Code

将 AI 回复中的代码块插入到编辑器中

上下文感知

自动包含当前 R 会话的包、变量、选中代码

模型指示器

底部显示当前使用的 LLM 模型名称

6. 代码解释

assistant_explain("mtcars %>% group_by(cyl) %>% summarise_all(mean)")

# 或在 RStudio 中选中代码后运行
assistant_explain()

# 控制详细程度
assistant_explain(code, detail_level = "brief")     # 简要
assistant_explain(code, detail_level = "detailed")  # 详细

7. 代码重构

assistant_refactor(
  "for(i in 1:nrow(df)) { df$x[i] <- df$y[i] * 2 }",
  style = "tidyverse",  # "base" / "data.table"
  goal  = "readability"  # "performance" / "conciseness"
)

8. 错误修复

# 选中报错的代码后运行（自动读取最后一条错误）
assistant_fix()

# 手动指定
assistant_fix(code = "df %>% filter(x = 5)",
             error = "could not find function '%>%'")

9. 代码与文档生成

# 从描述生成代码
assistant_complete("创建一个函数，返回正态性检验结果")

# 生成 roxygen2 文档
assistant_document("my_func <- function(x, y) { x + y }")

# 生成单元测试
assistant_test("add <- function(a, b) a + b")

12. 对话历史

assistant_history(n = 5)         # 最近 5 条
assistant_clear_history()       # 清空历史

保存在 ~/.r-assistant/history.json，默认最多 50 条。
